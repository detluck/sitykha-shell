# Quickshell Custom Desktop Environment

> ⚡ Currently under active development — starting with a lockscreen, but eventually including bar, launcher, settings app, and more.

This project is a **modular desktop environment built on Quickshell**. It aims to provide a fully configurable, lightweight, and visually appealing interface for Linux users.

---

## Current Focus

### LockScreen
- Customizable lock surface (background color, blur, brightness, saturation)
- Clock and date display with configurable fonts
- Avatar display and styling
- Password input with icon support
- Message area for prompts or notifications
- Tooltips for guidance
- JSON-based configuration with live reload support

---

## Planned Features

### Bar
- Workspace indicators and active window tracking
- System tray and status icons
- Media controls
- Quick toggles (Wi-Fi, Bluetooth, audio, brightness)

### Launcher
- Quick application access
- Action shortcuts
- Wallpaper previews
- Fuzzy search for apps and actions

### Settings App
- Real-time configuration editing
- Appearance, bar, launcher, and lockscreen settings
- Theme and font management

### System Integrations
- Power modes (performance, silent, energy-saving)
- Battery and fan management
- Notifications handling
- Multi-user support
- Fingerprint and other authentication options
